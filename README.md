# Corporate Credit Rating Engine

**Global Bank-Grade Internal Rating Model **

A lightweight, realistic corporate credit scoring tool to be used in wholesale/corporate banking.

### Features
- Calculates key financial ratios  
  → Leverage (Total Debt / EBITDA)  
  → Interest Coverage (EBITDA / Interest)  
  → EBITDA Margin  
  → Current Ratio  
- 5-factor scorecard with bank-style weights (30-25-20-15-10)  
- Internal rating: **AAA → D**  
- Recommended maximum facility limit (EBITDA × rating multiplier)  
- Suggested pricing spread in basis points (+80 bps to +800 bps)  

Can be used for credit analysts, risk teams, and portfolio monitoring.
